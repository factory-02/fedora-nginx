# Информация

SPEC-файл для создания RPM-пакета **nginx**.

## Установка

1. Подключить репозиторий **METASTORE**: `dnf copr enable metastore/fedora-nginx`.
2. Установить пакет: `dnf install nginx`.

## Авторы

- [**Kitsune Solar**](https://kitsune.solar/) - мейнтейнер.
